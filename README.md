# e-plantShopping

Đây là dự án cuối khóa cho ứng dụng Paradise Nursery - cửa hàng bán cây cảnh trực tuyến.

## Mô tả dự án
Ứng dụng này cho phép người dùng xem danh sách các loại cây, thêm vào giỏ hàng và quản lý số lượng sản phẩm trong giỏ hàng.

## Công nghệ sử dụng
*   React.js
*   Redux Toolkit
*   CSS
