# Emotion Detection Application

Đây là dự án cuối khóa tập trung vào việc xây dựng một ứng dụng web phát hiện cảm xúc sử dụng Python, Flask và IBM Watson NLP.

## Mô tả dự án
Ứng dụng cho phép người dùng nhập vào một đoạn văn bản, sau đó hệ thống sẽ phân tích văn bản đó để đưa ra các chỉ số cảm xúc (anger, disgust, fear, joy, sadness) và xác định cảm xúc chủ đạo (dominant emotion).

## Các tính năng chính
*   **Phân tích cảm xúc**: Kết nối với API IBM Watson NLP để xử lý văn bản.
*   **Giao diện web**: Sử dụng Flask để tạo giao diện người dùng đơn giản và trực quan.
*   **Đóng gói module**: Mã nguồn được tổ chức thành một package Python có thể tái sử dụng.
*   **Kiểm thử**: Đã bao gồm các unit test để đảm bảo tính chính xác của thuật toán phân tích.
